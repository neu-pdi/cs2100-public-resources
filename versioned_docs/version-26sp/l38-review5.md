# Final Exam Review and Interview Prep

It is up to the instructor how to hold this lecture.

## Final Exam Topics

Recommended: have students vote on topics they would most like to cover, and add an option to "spend the entire lecture on technical interview prep"

- Classes: constructors, methods, and attributes
- Lists, sets, and dictionaries
- Properties
- Abstract methods
- Inheritance
- UML diagrams
- Privacy
- Static
- Coupling / cohesion / encapsulation
- Iterator
- Comparable
- Decorator / Strategy / Observer / Data Pull
